# CCWater - 降雨量分析器

CCWater 是一款基于 Python 开发的复古街机风格降雨量数据分析工具。它不仅提供了直观的数据可视化功能，还集成了 AI 数据获取能力，旨在为用户提供一种既有趣又实用的地理气象数据探索体验。

## 基础用法

### 1. 启动程序
- **源码运行**: 执行 `python main.py`。
- **打包运行**: 直接运行打包后的可执行文件。
- *注：程序首次运行会自动创建 `CCWater` 文件夹并进行自我部署。*

### 2. 数据管理
- **导入文件**: 点击左侧“导入文件”按钮，加载符合格式的 `.txt` 数据。
- **文件卡片**: 左侧列表以卡片形式显示已加载的文件，点击卡片即可切换当前分析的数据集。
- **合并文件**: 点击“合并文件”可以按自定义优先级将多个数据集整合在一起。

### 3. 核心交互
- **柱状图可视化**: 中间区域动态显示各地区的降水量对比。
- **双击下钻**: 双击某个地区的柱子，可进入“明细模式”，查看其下属区域的详细数据。
- **拖拽筛选**: 
    - 可以将右侧“活跃区域”的卡片拖入“排除区域”来过滤数据。
    - 在明细模式下，同样支持对子区域进行拖拽筛选。
    - 支持直接从图表中的柱子拖拽到侧边栏进行快速筛选。

### 4. AI 数据链接
- 在左侧输入框输入地名（如“陕西”或“咸阳市”），点击“运行搜索”。
- 程序将通过 AI 自动获取该地区的详细降雨量数据并生成新的本地文件。

### 5. 界面设置
- **切换主题**: 支持“暗色模式”与“亮色模式”的一键切换。
- **清晰模式**: 开启后将移除复古扫描线和像素阴影，提供更清晰的现代视觉效果。
- **全屏切换**: 按 `F11` 进入/退出真全屏，按 `Esc` 退出全屏。
